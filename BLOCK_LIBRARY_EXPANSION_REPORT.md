## Theme-Aware Safeguards

All 50 blocks follow strict theme-awareness rules:

1. **Text colors**: All text uses `var(--site-text)`, `var(--site-muted)` — never hardcoded
2. **Backgrounds**: Cards use `var(--site-surface)`, sections use `var(--site-bg)` — no fixed colors
3. **Borders**: All borders use `var(--site-border)` — adapts to theme
4. **Accent**: Primary highlights use `var(--site-primary)` and `var(--site-primary-light)`
5. **Buttons**: All CTAs use `var(--btn-bg)`, `var(--btn-text)`, `var(--site-btn-radius)` — inherits site button style
6. **Typography**: Headings use `var(--site-heading-font)` — respects theme font choice
7. **No black-on-black**: Every block renders readable content in both light and dark themes
8. **No empty shells**: Every block ships with realistic French default content
